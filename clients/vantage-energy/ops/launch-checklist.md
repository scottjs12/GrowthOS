# Launch Checklist — Vantage Energy Advisory

> Client-specific launch progress tracker. See `09_growth-ops/launch-checklist.md` for the full 5-phase framework.

---

## Phase 1: Foundation

- [ ] Analytics tool installed and receiving events
- [ ] UTM conventions documented and shared
- [ ] Primary landing page deployed
- [ ] Pixels installed (per paid channels)
- [ ] Form submission tracking confirmed
- [ ] Baseline dashboard created

## Phase 2: First Campaign

- [ ] First ICP selected for campaign
- [ ] First channel selected
- [ ] Ad creative produced
- [ ] Campaign launched
- [ ] Full funnel validated (48-hour check)

## Phase 3: Expand

- [ ] Second ICP or channel launched
- [ ] Email sequences activated
- [ ] Social channel started

## Phase 4: Optimize

- [ ] First LP A/B test running
- [ ] Email sequence optimization
- [ ] Ad creative refresh

## Phase 5: Scale

- [ ] All ICP landing pages deployed
- [ ] 3+ channels running
- [ ] Referral program launched (if applicable)
