# Weekly Operating Rhythm

> The cadence that compounds. Miss a week and you lose momentum. Miss two and you're starting over.

## Trigger
- Every Monday, automatically. This is the heartbeat of the system.

## Monday — Review + Plan (1 hour)

### Analytics (20 min)
- Publish weekly dashboard by 9am
- Flag any metric in 🟡 Yellow or 🔴 Red status
- Note any anomalies or unexpected trends
- Reference: `00_shared/reporting/weekly-dashboard-template.md`

### PM (20 min)
- Review dashboard for G/Y/R status across all metrics
- Check experiment status: anything closing this week? Anything blocked?
- Review agent task queue: any deadlines at risk?
- Set 3 priorities for the week

### Quick Actions (20 min)
- Paid: Adjust budgets based on last week's performance (+25% winners, pause losers)
- CRM: Check ESP for delivery issues, bounces, unsubscribe spikes
- Ops: Check lead queue — any MQLs waiting for routing?
- Social: Reply to weekend comments/DMs

## Tuesday — Create (2 hours)

### Content Production Block
- Content Writer: Batch-produce week's content (social posts, email copy, ad refreshes)
- Designer: Produce any creative assets needed for the week
- SEO: Publish any content that's been in review
- Rules: Batch everything. Context-switching kills quality.
- Reference: `workflows/content-production.md`

## Wednesday — Build + Ship (1 hour)

### Execution Block
- Engineer: Ship any builds in queue (LP updates, tracking fixes, test instrumentation)
- Ops: Traffic any new campaigns or creative swaps
- Paid: Launch any new ads or audience tests queued from Monday
- CRO: Review any test results reaching statistical significance

## Thursday — Analyze (30 min)

### Mid-Week Check
- Analytics: Mid-week pulse — any campaigns drifting? Any anomalies since Monday?
- Paid: Check frequency, CPA trends, creative fatigue signals
- CRM: Check email flow performance (especially new sequences in first 2 weeks)
- Flag anything that needs Friday action

## Friday — Close + Prep (30 min)

### Wrap-Up
- PM: Update experiment tracker status
- Ops: Weekly snapshot filed to `00_shared/reporting/weekly-snapshots/`
- Researcher: Queue any community listening or competitive intel for next week
- PM: Flag any strategic decisions needed for Monday sync

## Total Weekly Time: ~4.5 hours

## Escalation Triggers (any day)
- KPI >20% off target for 3+ consecutive days → PM escalates to Strategist
- Creative fatigue detected → Trigger `workflows/creative-refresh.md` immediately
- Tracking broken → Ops + Engineer fix within 4 hours, Paid pauses spend
- Budget overrun >25% → Paid pauses, PM + Strategist review

## Rules
- Monday dashboard is non-negotiable — the week doesn't start without it
- Tuesday is creation, not meetings. Protect the block.
- Never skip Friday close — next Monday's plan depends on it
- If you're under 3 hours/week, you're skipping something important
- If you're over 6 hours/week, you're not batching efficiently
