# Creative Refresh Workflow

> When ad creative gets stale, this is how you respond. Speed matters — every day of fatigue is wasted spend.

## Trigger
- Paid flags creative fatigue: Frequency >3.5 OR CTR drops >20% while CPM holds
- Scheduled refresh: Every 2-3 weeks for active campaigns
- Post-test routing: Winning angle needs new variants to scale

## Agent Chain

STEP 1: Paid — Fatigue Signal + Brief
- Input: Performance data showing fatigue indicators
- Output: Fatigue alert with: which creative is fatiguing, current metrics, audience affected, angle direction for refresh
- Timeline: Flag within 24h of detecting fatigue signal
- Handoff: Alert → Content Writer + Designer

STEP 2: Content Writer — New Copy Angles
- Input: Fatigue alert + winning angle data + ICP context
- Output: 3+ new copy variants (keep winning angle, change hook/framing)
- Rules: Don't abandon the winning angle — iterate on it. New hooks, same core message.
- Timeline: 24h turnaround
- Handoff: Copy → Designer (structured format)

STEP 3: Designer — New Creative Variants
- Input: New copy + existing brand assets + performance data on what worked visually
- Output: Min 2 static + 1 motion variant per copy direction
- Rules: Evolve the winning visual approach — don't start from scratch. Small changes often beat big pivots.
- Timeline: 48h turnaround from receiving copy

STEP 4: Ops — Quick QA
- Input: New creative assets + UTMs
- Output: Verified naming conventions, UTMs, asset specs match platform requirements
- Rules: Abbreviated checklist — focus on naming, UTMs, and asset specs only (tracking already confirmed from original launch)
- Timeline: Same day

STEP 5: Paid — Swap + Monitor
- Rotate new creative into existing campaigns within 24h of Ops sign-off
- Pause worst-performing old creative (don't kill all — keep 1 control)
- Monitor new creative for 48h before adjusting budgets

## Total Timeline: 4-5 days from fatigue signal to new creative live

## Exit Criteria
- [ ] New creative live in platform
- [ ] Old fatigued creative paused (1 control kept)
- [ ] Copy bank updated with new variants
- [ ] 48h performance baseline for new creative documented

## Rules
- Never let fatigue run more than 7 days — that's wasted budget
- Always keep one control creative running for comparison
- Don't abandon winning angles — fatigue is about execution (visuals, hooks), not strategy
- Log refresh cycle in copy bank for future timing reference
