# Weekly Growth Dashboard Template

> The Monday report. One page, every number that matters. If it's not on this dashboard, it's not a priority this week.

## How to Use

1. Fill in the KPI rows during client onboarding (match to growth model)
2. Update every Monday morning before the weekly review
3. Green/Yellow/Red status drives the week's priorities
4. Archive each week's snapshot in `00_shared/reporting/weekly-snapshots/`

## Status Definitions

| Status | Meaning | Action |
|---|---|---|
| **Green** | On or above target | Continue current approach |
| **Yellow** | 10-20% below target | Investigate, adjust this week |
| **Red** | >20% below target | Escalate to Strategist, reprioritize |
