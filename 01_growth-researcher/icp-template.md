# ICP Profile Template

> Universal template for defining Ideal Customer Profiles. Complete one per ICP segment. Minimum 2, maximum 5 per client.

## Instructions

1. Copy this template for each ICP segment
2. Fill in every field — leave nothing as "[placeholder]" before routing to other agents
3. Be specific: "35-year-old ops manager at a 50-person SaaS company" beats "business professional"
4. Validate with real data: interviews, surveys, analytics, or sales call transcripts — not assumptions
5. Route completed profiles to: Content Writer (messaging), Paid (targeting), CRM (segmentation), Designer (visual tone)
