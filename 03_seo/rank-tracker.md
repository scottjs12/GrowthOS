## How to Use

1. Add your target keywords from `keyword-universe.md`
2. Record rankings weekly (same day each week for consistency)
3. Track trend: ↑ improving, ↓ declining, → stable
4. Priority: P1 (money keywords), P2 (traffic drivers), P3 (long-tail)
5. Flag any keyword that moves 10+ positions in either direction — investigate immediately

## Ranking Sources

| Tool | Best For |
|---|---|
| Google Search Console | Actual impression/click data, average position |
| Ahrefs / SEMrush | Competitor comparison, SERP feature tracking |
| Manual search (incognito) | Quick spot-check, SERP layout context |

**Rule:** Never rely on a single source. Cross-reference GSC position data with rank tracker tool data monthly.
