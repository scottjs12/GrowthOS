## Status Key

| Status | Meaning |
|---|---|
| `queued` | Brief received, not started |
| `building` | In development |
| `preview` | Preview URL ready for review |
| `qa` | Ops running trafficking checklist |
| `live` | Deployed to production |
| `iterating` | Live but being modified based on test results |
