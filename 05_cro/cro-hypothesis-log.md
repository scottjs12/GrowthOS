## Status Key

| Status | Meaning |
|---|---|
| `backlog` | ICE-scored, waiting for scheduling |
| `briefed` | PM has issued the brief |
| `building` | Engineer is implementing the variant |
| `live` | Test is running — do not touch |
| `closed` | Analytics has called the result |
| `archived` | Learning captured, no follow-up needed |

## Result Key

| Result | Meaning |
|---|---|
| `win` | Variant beat control at 95% confidence |
| `loss` | Control beat variant at 95% confidence |
| `inconclusive` | No significant difference detected |
| `invalid` | Test was compromised (confound, tracking issue, insufficient sample) |
